Creating Your First Web Page!

Let’s create your first ever webpage. First you will need a text editor to code out HTML. We would recommend that you install atom.io as your text editor to code HTML as it is free and easy to use. Next, install and open atom. You will find a Make save a file to your computer with the extension “.html” and get ready to code!

For a start, we will need to use the <html> tag to allow the text editor know that we are using HTML from that point on. The HTML tag must be finished off by another HTML tag like this. 
 <html>
	<!-- Your code would go here. This “<!-- -->” tag denotes a comment that will not be shown on the website.-->
</html>

Next we will add the <head>. This tag links files and provides extra information about the webpage. Your webpage should now look like this: 

<html>
	<!-- The 'head' tag should be contained inside the 'html' tag, as all the HTML code goes inside here... --&t;
	<head>
		<!-- This is where the content in the 'head' code goes -->
	</head>
</html>

Finally we will add the <body> tag. This shows what will be displayed on the webpage. Your webpage should now finally look like this:

<html>
	<!-- The 'head' tag should be contained inside the 'html' tag, as all the HTML code goes inside here... -->
	<head>
		<!-- This is where the content in the 'head' code goes -->
	</head>
</html>

Common HTML tags include the: <!DOCTYPE> Defines the document type <html> Defines an HTML document <head> Defines information about the document <title> Defines a title for the document <body> Defines the content that is to be displayed <h1> to <h6> Defines a heading <p> Defines a paragraph <br/> Inserts a single line break So without further ado let’s get something in your body to show up on the webpage. First let’s add a title to your webpage. Next let’s add a heading to your webpage as well as a paragraph. Your .html file 

<html>
	<head>
		<title>My First Ever Web Page!</title>
	</head>
	<body>
		<h1>Hello World! Welcome to My First Ever Website!</h1>
		<p>This is a website made by finely coded, pure HTML. (You can add any other text you want...)</p>
	</body>
</html>
